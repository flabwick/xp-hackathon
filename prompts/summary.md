# Course Scope Summary

## ✅ Included Content
- **Logic & Set Foundations**: Propositional connectives, truth tables, material implication, proof formalization. Universal/existential quantifiers, nested operators, De Morgan’s laws, natural language translation. Set-builder syntax, union/intersection/complement, Cartesian products, orthants, feasible region notation. Functions, composition, injectivity/surjectivity, monotonicity, argmax invariance.
- **Real Analysis & ℝⁿ Topology**: ℝ field axioms, trichotomy, absolute value, triangle inequality, Archimedean property, density of ℚ. Bounds, least upper bound property, sup/inf existence, attainment conditions. ℝⁿ vectors, Euclidean inner product, Cauchy-Schwarz, norms, Pythagorean theorem. Metric axioms, open/closed balls, interior/boundary/exterior. ε-N convergence, limit arithmetic, uniqueness. Cauchy criterion, ℝⁿ completeness, subsequence extraction, Bolzano-Weierstrass. Open/closed sets, closure/interior/boundary operators. Sequential/open-cover compactness, Heine-Borel theorem. ε-δ/sequential continuity, preimage of open sets, algebra of continuous functions, Extreme Value Theorem.
- **Linear Algebra & Multivariate Calculus**: Matrix operations, transpose, dimensional compatibility, rank equivalence, determinants, invertibility, adjoint/cofactor construction. Single-variable limits, continuity, difference quotients, local linear approximation, MVT. Total derivative via affine approximation, C¹ classification, differentiation rules. Partial/directional derivatives, gradient vector, total derivative identity, C¹ sufficiency. Hessians, Clairaut’s theorem, C² class, second-order Taylor expansions. Quadratic forms, symmetry, definiteness taxonomy, Sylvester’s criterion.
- **Optimization Theory (Unconstrained & Constrained)**: Max/min notation, objective/constraint separation, interior/boundary feasibility, argmax/argmin sets. FOC derivation via limits, interior optimality necessity, critical point vs optimum distinction. Hessian SOCs, eigenvalue/minor tests, local extrema classification. Semidefinite edge cases, inconclusive tests, boundary comparison. Feasible manifolds, tangent spaces, geometric gradient alignment. Lagrangian formulation, DL=0 derivation, multiplier signs. Inequality geometry, active/slack classification, complementary slackness, case enumeration. KKT necessary conditions, stationarity, non-negativity, mixed constraint unification. LICQ, rank conditions, constraint qualification diagnostics. Multiplier sensitivity, shadow prices, parametric relaxation. Restricted quadratic forms, bordered Hessians, determinant sign rules. Weierstrass existence, compactness/continuity requirements.
- **Convexity, Parametric Analysis & Synthesis**: Convex combinations, line-segment containment, preservation properties. Epigraphs/subgraphs, convexity via set property, sub/super-level geometry. Jensen’s inequality, concave/convex duality, tangent hyperplane support, gradient characterization. Interior continuity, one-sided limits, directional derivative existence, negligible kink sets. Strict concavity/convexity, affine boundaries, local-to-global equivalence, singleton collapse. Global sufficiency, gradient inequality collapse, SOC bypass. Quasi-concavity/convexity, upper/lower contour sets, critical point sufficiency. Value function definition, parameter shift propagation, structural dependency. Smooth interior envelope theorem, indirect effect cancellation. KKT/corner envelope extensions, multiplier contributions. Correspondences, graph properties, upper/lower hemicontinuity, compact/convex-valued preservation. Berge’s Maximum Theorem, continuity of value/solution maps, single-valuedness conditions. Utility/expenditure & profit/cost duality, Pareto optimality, welfare equivalence. Full proof pipeline execution, assumption auditing, necessity vs sufficiency architecture, mathematical-to-economic translation.
